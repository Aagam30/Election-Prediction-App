## 🧠 Model Overview

The election predictor uses a supervised machine learning model trained on historical demographic and election data. Key features include:

- 📅 Age  
- 🎓 Education Level  
- 🗳️ Caste Category  
- 🏛️ State  
- 🆔 Simulated Voter ID stats
